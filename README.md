## Présentation

Ce site est un **poisson d’avril** — une blague numérique — lancé en 2024.  
Le thème : une fausse agence “sith wéb professionnels” promettant de “réaliser votre image en ligne” avec des slogans volontairement mal écrits et des promesses absurdes.  
Le but est de surprendre les visiteurs, de jouer sur le langage, et de faire rire tout en montrant tes compétences techniques via le site (animations, mise en page, etc.).

Accès au site : [el-cassegrain.github.io/poisson-avril-2024](https://el-cassegrain.github.io/poisson-avril-2024/)

## Contexte / objectif

- Sortir un poisson d’avril créatif, différent de ce qu’on voit d’habitude  
- Montrer ta capacité à utiliser le web pour créer une illusion / performance artistique  
- S’amuser (et potentiellement embêter un peu les visiteurs)  
- Servir de pièce de portfolio sur l’usage de typographie, animation, textes “déchirés”, etc.

## Fonctionnalités / contenu

- Page d’accueil avec slogan “CRéation de sith wéb professionnels”  
- Menu / lien “Contact” (pour le semblant de légitimité)  
- Effets visuels ou animations (si présentes)  
- Texte volontairement mal orthographié pour l’effet comique  
- Structure simple, chargement rapide
